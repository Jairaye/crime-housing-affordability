\# Crime + Housing Affordability (Metro Level)



Goal: evaluate where a $100K household is and isn't enough, comparing renting vs buying across U.S. metro areas.



\## Data Sources

\- Zillow ZHVI (Single-Family, Mid-tier, Metro)

\- Zillow ZORI (Rent, Metro)

\- U.S. Census Metro Population (CBSA)



\## Project Structure

\- data/raw: source downloads (not committed)

\- data/processed: analysis-ready outputs

\- notebooks: analysis workflow

\- src: reusable scripts

\- outputs: charts/tables (not committed)
